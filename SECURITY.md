# Security Policy

## Supported Versions

Currently, the following versions of Ollama Vox are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.1  | :white_check_mark: |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues, discussions, or social media.

Instead, please report them directly to the maintainers using GitHub's Private Vulnerability Reporting feature:

1. Go to the [Security tab](https://github.com/maadhav-codes/ollama-vox/security/advisories/new) of the repository.
2. Click on **Report a vulnerability**.
3. Fill in the details of the vulnerability, including steps to reproduce, and submit it.

The maintainers will review the report and respond privately as soon as possible.
